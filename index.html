Python Module Installer

Automatically install missing Python modules from your script with ease!

Overview

This Python script helps you to automatically detect and install missing dependencies from your Python script. It scans for import statements, extracts the necessary modules, and installs them for you.

Features

  • Auto scans Python scripts for import statements
  • Installs missing modules using pip
  • Supports drag-and-drop and file path input
  • Skips built-in Python modules
  • Fast and efficient!

2000+ Supported Modules

This script supports over 2000 popular Python modules, including:

  • numpy
  • pandas
  • matplotlib
  • scipy
  • requests
  • flask
  • tensorflow
  • beautifulsoup4
  • sqlalchemy
  • pytorch
  • pytest
  • django
  • seaborn
  • scikit-learn
  • pytest-django

How to Use

  1. Clone or download the repository from GitHub.
  2. Run the script: 
    python module_installer.py
  3. Provide your Python script's file path when prompted (or drag and drop the file).
  4. The script will automatically detect and install any missing modules.
